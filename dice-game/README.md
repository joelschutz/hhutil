# Dice

A dice roller application for handheld game consoles made with Love2d that support most used dice types.

## Features 

- Support most used dice(d4, d6, d8, d10, d12, d20, d100 and coin flip)
- Can roll up to 8 dice simultanuasly
- Critials are colored for extra impact!

## Controls
| Button | Action |
|--|--|
|D-Pad Left/Right|Navegate Dice|
|D-Pad Up/Down|Change Die|
|A|Add Die|
|B|Remove Selected|
|Y|Roll Selected|
|X|Roll All Dice|
|Select|Open Credits|
|Start|Change Pallete|

# Habit

A habit tracker application for handheld game consoles made with Love2d.

## Features

- Track binary or numeric habits
- Weekly, monthly, and yearly views
- Gamepad and keyboard controls
- Persistent data storage

## Controls

| Button | Action |
|--|--|
|D-Pad Left/Right|Navigate days|
|D-Pad Up/Down|Navigate rows|
|LB/RB|Switch habit|
|A|Annotate selected day|
|B|Cycle visualization mode|
|X|Edit current habit|
|Y|Create new habit|

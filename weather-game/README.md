# Weather

A weather application for handheld game consoles made with Love2d that can detect your location.

## Features and limitations

- Shows forecast for 7 days and current day
- Caches your added cities for convenience
- Can detect your location automaticaly
- Shows current Temperature, Humidity and Weather
- Shows forecast Weather, Min/Max Temperature, Precipitation Probability and Volume
- Adds new cities by coordinates
- Time and Date included!

**Limitations**:
- Internet access is mandatory
- Location by IP may not be 100% accurate
- Not all Weather Codes are supported

## Controls
| Button | Action |
|--|--|
|D-Pad Left/Right|Navegate Cities|
|A|Refresh Forecast|
|B|Day Forecast|
|Y|Add City|
|X|Remove City|
|Select|Open Credits|
|Start|Change Pallete|

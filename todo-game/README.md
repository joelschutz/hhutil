# TODO List App

A minimalist TODO list application for handheld consoles, built with LÖVE 2D.

## Features

- Add, edit, delete, and complete TODO items
- Multiple color palettes 
- Persistent storage

## Controls

| Button | Action |
|--------|--------|
| **D-Pad Up/Down** | Navigate TODO list (wraps at ends) |
| **Y** | Add new TODO (enters edit mode) |
| **X** | Delete selected TODO |
| **A** | Toggle TODO completion status |
| **B** | Edit selected TODO |
| **START** | Cycle color palettes |
| **SELECT (Back)** | Show/hide credits |

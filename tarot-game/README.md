# Tarot

A tarot application for handheld game consoles made with Love2d to read your future.

## Features

- Includes the Major Arcana Cards
- Read your Past, Present and Future
- Pointer for improved reading
- Candles included!

## Controls
| Button | Action |
|--|--|
|D-Pad|Navegate Pointer|
|A|Reveal Card|
|B|Close Cards|
|Select|Open Credits|
|Start|Change Pallete|

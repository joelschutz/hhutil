# Metronome

A metronome application for handheld game consoles made with Love2d that includes 10 different clicks.

## Features 

- Support beat and bar count
- Has visual indication for better use
- Minimal interface
- Multiple click sounds to choose!

## Controls
| Button | Action |
|--|--|
|D-Pad Left/Right|Change Bar Count|
|D-Pad Up/Down|Change BPM|
|A|Play|
|B|Stop|
|Y/X|Change Sound|
|Select|Open Credits|
|Start|Change Pallete|
